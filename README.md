# radioamin-android

Android Client App For [My Training Java-BackEnd Project](https://github.com/mabahmani/radioamin-back-java) 

Written in Kotlin With MVVM Arch components and Coroutine

You Can Play Your Local Music Files If you have not logged in to app

Also You Can Edit Your Local Music Files
